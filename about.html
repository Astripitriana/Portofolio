Profil

Halo! Saya Astri Fitriana Mustika, atau biasa dipanggil Astri. Saat ini berdomisili di Bandung. Saat ini saya adalah mahasiswa aktif semester 6 di STMIK Mardira Indonesia, Program Studi Teknik Informatika dengan konsentrasi Sistem Informasi.

Saya mulai tertarik dengan dunia pemrograman sejak duduk di bangku SMK, dengan mempelajari dasar-dasar Object Oriented Programming (OOP) menggunakan PHP dan Java. Seiring waktu, saya terus mengembangkan keterampilan melalui berbagai bahasa dan framework, seperti Laravel untuk pengembangan web backend, serta Flutter & Dart untuk membangun aplikasi mobile multiplatform.

Project pertama yang saya kerjakan adalah Sistem Informasi Sekolah berbasis web menggunakan PHP, Bootstrap, dan MySQL. Saat ini, saya sedang fokus mendalami pengembangan web dan membangun aplikasi, baik untuk desktop maupun mobile, sambil terus belajar agar bisa beradaptasi dengan teknologi baru.

Di dunia perkuliahan, saya aktif dalam organisasi UKM Creative Student Association sebagai anggota divisi Pemrograman dan pernah menjabat sebagai staf Penelitian dan Pengembangan. Semasa SMK, saya juga pernah dipercaya sebagai Bendahara OSIS, yang mengasah kemampuan saya dalam tanggung jawab, pengelolaan data, dan kerja tim.

Jika anda tertarik mengenal lebih jauh tentang saya, silakan lihat CV saya atau hubungi melalui media sosial yang telah saya cantumkan di web ini. Terima kasih sudah berkunjung!
